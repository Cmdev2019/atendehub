# 🗄️ Arquivo histórico

Relatórios de status e documentos de sessões antigas de desenvolvimento,
movidos da raiz em 2026-07-16 (item F5-1 do roadmap).

> ⚠️ **Não use estes documentos como fonte de verdade** — vários afirmam mais
> do que o código entregava na época (ex.: `SLA_FIXES.md` dizia "pronto para
> deploy" com o módulo desativado; `TESTE_REDIS_ADAPTER.md` descreve um adapter
> que nunca chegou a ser aplicado — ver F7-5). O estado real e atual do projeto
> vive em [`ROADMAP_ESTABILIZACAO.md`](../../ROADMAP_ESTABILIZACAO.md) e o
> contrato da API em [`API_CONTRACT.md`](../API_CONTRACT.md).
